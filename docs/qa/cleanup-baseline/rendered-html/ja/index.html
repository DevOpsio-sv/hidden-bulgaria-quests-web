使い方

ひとつの場所から、 守護者へ。

Unlocking Bulgaria のすべての旅は明確な進行をたどる——単一のチェックポイントから、ブルガリアの秘められた記憶の完全な解放まで。

  1. 01

    ステップ 1

    チェックポイントを訪れる

    実際の場所へ向かう。到着することでチェックポイントを起動する。ストーリー、手がかり、または隠されたレイヤーを解放する。

  2. 02

    ステップ 2 — 5チェックポイント

    ルートを完了 → 遺物を収集

    5つの連結したチェックポイントが1つのルートを形成する。ルートを完了すると、永続的なデジタル遺物——ブルガリアの歴史の遺産を受け取る。

  3. 03

    ステップ 3 — 3ルート

    領域を解放 → 封印を獲得

    3つのルートが地域の領域を解放する。その封印を獲得する——その地域の秘められた記憶を制した証。

  4. 04

    ステップ 4 — 6領域

    国家の解放 → 守護者になる

    すべての6つの領域を完了する。ブルガリアの完全な記憶を解放する。守護者——Пазитель の称号を受け取る。

681

国家

ハン・アスパルフがブルガリア国家を建国する。

ドナウ川のほとりで、ブルガール人とスラヴ人の連合体が最初のブルガリア国家を形成する——決定的な軍事的勝利の後、ビザンチン帝国に認められた。ブルガリアという名が歴史に刻まれる。これ以降のすべてを生き抜く名前。

862

文字

スラヴ民族のための新しいアルファベットが生まれる。

テッサロニキから来た二人の兄弟がグラゴル文字を創作する——後のキリル文字の基盤となった。初めて、スラヴ世界は自分たちの文字を持つことになる。ブルガリアはこの遺産を何世紀にもわたって採用し保存する。

865

洗礼

ハン・ボリス1世がブルガリアのためにキリスト教を受け入れる。

深い文化的影響を持つ計算された政治的行為として、ハン・ボリス1世はブルガリアをキリスト教に改宗させる。信仰と識字を中心とした新たなアイデンティティが形成される。プリスカとプレスラフはスラヴ・キリスト教文明の中心地となる。

893

言語

古ブルガリア語が国家と教会の公用語となる。

ツァーリ・シメオン1世のもと、プレスラフ文学学校が花開く。古ブルガリア語がすべての公式な場でギリシャ語に取って代わる。ブルガリアは中世ヨーロッパの中心で文学的・精神的な大国となる。

1018

崩壊

ブルガリアは倒れる——だが、名は生き続ける。

バシレイオス2世が数十年の戦争の末にブルガリアを征服する。ブルガリア国家は167年間存在しなくなる。しかし、名前、言語、記憶は生き残る——修道院に、村に、口承伝統に、石そのものの中に守られ続けて。

1185

帰還

第二次ブルガリア帝国がタルノヴォの丘から立ち上がる。

アセンとペタルの兄弟がタルノヴォの丘からの反乱を指揮する。ブルガリアは再生する——完全な征服の後に自らを再建した唯一の中世スラヴ国家。名前は生き残った。記憶は戻った。旅は続く。
