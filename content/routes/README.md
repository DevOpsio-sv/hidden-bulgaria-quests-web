# Content / Routes

This folder contains structured content planning files for individual routes within each domain.

## Route Structure

Each route contains:
- Exactly **5 checkpoints**
- 1 **Relic** awarded on completion
- Narrative arc connecting the 5 locations
- Historical or thematic thread

## Route ID Convention

`route_{descriptor}` — lowercase snake_case

### Example Route IDs
```
route_northern_black_sea
route_danube_fortresses
route_old_capitals_core
route_thracian_tombs
route_rhodope_voices
route_sofia_layers
```

## Intended Contents (Per Route)

- `{route_id}.md` — route overview, narrative concept, checkpoint list, relic description
