# Design / Assets Guidelines

This folder contains technical specifications and guidelines for all media assets used in Unlocking Bulgaria.

## Intended Contents

- Image size and format requirements (hero, domain covers, checkpoint thumbnails, relics)
- Video encoding specifications (domain cinematics, checkpoint reveals)
- Audio format requirements (narration, ambient, AR soundscapes)
- AR effect asset specifications (textures, overlays, animation files)
- File naming conventions for assets
- Delivery pipeline for media assets (how to prepare and submit)

## Asset Naming Convention

Follow the conventions in `/docs/governance/NAMING_CONVENTIONS.md`:

```
{type}-{descriptor}-{size}.{ext}

Examples:
hero-bg-1920.jpg
domain-black-sea-cover-800.jpg
checkpoint-kaliakra-thumb-400.jpg
relic-sea-memory-400.png
ar-overlay-guardian-sight.svg
checkpoint-kaliakra-narration-en.mp3
```

## Current Asset Locations

- `public/` — Static site assets (favicon, SEO files)
- `public/_assets/` — Generated by Astro build (CSS, JS bundles)
